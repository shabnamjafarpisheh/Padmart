# 🎭 Padmart Acting School Reservation System

A modern Streamlit-based reservation platform for acting school studios and rehearsal halls.

## Features

### User Features
- User Registration
- Secure Login
- Hall Reservation System
- Live Availability Calendar
- Reservation History
- Cancel & Refund Reservations
- Password Management
- English / فارسی Support

### Owner Dashboard
- Full Reservation Management
- Book on Behalf of Guests
- Availability Calendar
- Reservation Analytics
- Revenue Tracking
- User Management
- Suspend / Activate Users
- Delete Users
- Search Users
- Hall Utilization Statistics

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
